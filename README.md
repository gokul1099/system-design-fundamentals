## 🌐 Live Course

**[Open the course → https://gokul1099.github.io/system-design-fundamentals/](https://gokul1099.github.io/system-design-fundamentals/)**

## 📚 What's Inside

Each of the 103 topics includes:

- **Core Concept** — original course content, cleaned up
- **Beginner Analogy** — everyday comparisons to make concepts click
- **Real-World Examples** — how Netflix, Google, Uber, Amazon, and others use these concepts
- **Architecture Diagram** — visual ASCII diagram for every topic
- **Interview Tips** — what interviewers look for and common mistakes to avoid
- **Key Takeaways** — concise summary for quick revision
- **Further Reading** — curated links to official docs and authoritative resources

## 🚀 Run Locally

Just open `index.html` in any browser — no server needed, works fully offline.

```bash
open enhanced-course/index.html
```

## 🛠️ Rebuild

```bash
cd /path/to/fund
python3 build_course.py
```

Requires Python 3 only — no dependencies.
